# TMB Business Requirements Document (BRD) — Phase 0 Summary

## Client
Toyloq District Administration (Hokimlik), Samarkand Region, Uzbekistan

## Project
Education Monitoring & Rating Platform (TMB)

## Objectives
1. 100% digitalization of non-budget education sector
2. Data-driven administrative decisions
3. Transparent, tamper-evident center ratings
4. Public trust via certificate verification
5. Multi-tenant architecture (district → region → republic)
6. Security as strategic objective

## Phase 0 Scope (Delivered)
- Architecture and threat model
- Identity/auth subsystem (8 roles)
- Database schema foundation
- i18n skeleton (UZ/RU/EN)
- CI/CD scaffold

## Stakeholders
- District IT (Super Admin)
- Hokimiyat operators
- Center directors and admins
- Teachers, auditors
- Parents/guardians
- Regional statistics committee (external API)
