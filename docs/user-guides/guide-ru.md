# TaMoR User Guide — Русский (RU)

## Вход в систему

1. Откройте `http://localhost:3000/login` в браузере
2. Выберите язык через переключатель `UZ | RU | EN` в правом верхнем углу
3. Введите имя пользователя и пароль
4. Для аккаунтов с MFA введите код из Google Authenticator

Войдите по номеру телефона (+998901234567) с SMS-кодом. В режиме разработки код отображается в консоли сервера.
