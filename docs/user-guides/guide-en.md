# TaMoR User Guide — English (EN)

## Signing In

1. Open `http://localhost:3000/login` in your browser
2. Select your language via the `UZ | RU | EN` switcher in the top-right corner
3. Enter your username and password
4. For MFA-enabled accounts, enter the code from Google Authenticator
